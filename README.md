# savemz 

# It's easly save with json file in python 

## install library

```bash
pip install savemz
```

## how to use?
```python
from savemz import SaveMZ

db = SaveMZ("file_name.json")

db.set("users.name", "Julius")
#To write data in file using easy path

db.get("users.name")
#To get data from file

db.reload()
#To reload cache and file
```
